## 优先级定义

- **Critical (关键)**: 影响核心功能，必须立即实施
- **High (高)**: 重要功能，近期需要实施
- **Medium (中)**: 有价值的增强功能，中期规划
- **Low (低)**: 锦上添花的功能，长期考虑

---

## 状态定义

- **Backlog**: 待排期，已评估
- **Planned**: 已排期，即将开始
- **In Progress**: 开发中
- **Testing**: 测试中
- **Done**: 已完成
- **Cancelled**: 已取消
